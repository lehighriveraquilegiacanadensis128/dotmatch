# Support

Use GitHub issues for reproducible bugs, documentation problems, feature requests, and benchmark questions.

## Useful Issue Contents

- DotMatch commit or release.
- Operating system and CPU.
- Install method: source, Docker, Python, or other.
- Exact command and input schema.
- Small synthetic input that reproduces the behavior.
- Expected result and observed result.

For benchmark questions, include the raw CSV row, comparator version, exact command, and which evidence gate or report is affected.

Do not post private FASTQ/BCL files, human subject data, unpublished datasets, or proprietary sample sheets in public issues.
